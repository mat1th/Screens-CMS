## Installation

Download: <https://github.com/mat1th/meesterproef.git>

Go to the project:

```bash
cd path/to/files
```

Install node modules:

```bash
npm install
```

## Building

Install Gulp:

```bash
npm install --global gulp-cli
```

Go to the project:

```bash
cd path/to/files
```

Install dev npm modules:

```bash
npm install --dev
```

Start Gulp:

```bash
gulp
```
